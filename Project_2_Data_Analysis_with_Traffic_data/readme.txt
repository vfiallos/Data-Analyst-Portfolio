Software Required: Latest version of Jupyter Notebook

Please follow the below instructions to run the Notebook file.

Step 1: The main file to run is called Finding Heavy Traffic Indicators on I-94.ipynb
		 
	First upload the datasets Metro_Interstate_Traffic_Volume.csv.gz. Finding Heavy Traffic Indicators on I-94.ipynb file is already present in notebook folder do not change the
        path of anyfile. Just open the folder in Jupyter Notebook and start running each cell of Finding Heavy Traffic Indicators on I-94.ipynb. 

Step 2: If required package is not installed then first install the required package which is used in this project.


Step 3: You can run the notebook document step-by-step (one cell a time) or you can run the whole notebook in a single step by clicking on the menu cell-> Run All.
